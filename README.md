# Visionário Frases

Projeto de frases motivacionais com sistema de administração.

## Tecnologias utilizadas

- **Frontend**: HTML, CSS, JavaScript (com um pouco de TypeScript)
- **Backend**: Python + Flask + SQLite
- **Autenticação**: Flask-Login

## Funcionalidades

- Visualização de frases
- Login de administradores
- Adicionar novas frases (apenas admins)
- Deletar frases (apenas admins)
- Design responsivo
